# Fetch Temperature

## A simple app to fetch the temperature with the postal code of the location.

### Objective:

This project was developed as a personal challenge motivated by curiosity, aiming to improve my skills in itegrating ,consuming,and manipulating different external APIs.

### To start the server:

use "npx tsx src/index.ts" to start the backend server,
use "npm run dev" to start the frontend server

### Usuability:

Type the postal code of the location you want to see the local temperature.
